⚛️ MANUAL DE OPERACIÓN BWR-3000 ⚛️

OBJETIVO: Mantener estabilidad térmica y nuclear. Temperatura < 305°C, Presión < 1150 PSI, Nivel agua 40-85%.

🔧 PASOS PARA OPERACIÓN SEGURA:

  • 1. CONTROL RODS: Retira barras (OUT) para aumentar potencia. Inserta (IN) para reducir. Vigila APRM/IRM.
  • 2. RECIRCULATION SYSTEM: Ajusta Pump Speed para controlar flujo de núcleo (Core Flow).
  • 3. LEVEL CONTROL: Activa START PUMP y ajusta SETPOINT para mantener nivel agua estable.
  • 4. TURBINE/GENERATOR: Cierra BREAKER para generar electricidad. Usa GOVERNOR para regular RPM.
  • 5. REACTOR MODE: Inicia en STARTUP → RUN. Si todo falla, SHUTDOWN o SCRAM de emergencia.
  • 6. EVENTOS ALEATORIOS: Bombas fallan, picos de presión. Reacciona rápido para evitar SCRAM automático.

⚠️ ALARMAS CRÍTICAS: presión > 1100 PSI o nivel < 20% puede causar MELTDOWN.

BOTONES ACTIVOS: Raise/Lower setpoint, Start/Stop pump, In/Out rods, Governor, Breaker, Recirc Raise/Lower, SCRAM.
